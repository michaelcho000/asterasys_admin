### 2.1 누적 판매 vs 월간 판매의 차이

```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

─────────────────────────────────────────────

─────────────────────────────────────────────
올리지오   1,220대      188대        15.4% ⭐

중대 발견:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 총 판매량 = 누적 데이터
   - 리프테라 495대 = 출시 이후 전체 판매
   - 8월 마케팅 활동 성과 아님

2. 8월 신규 판매 = 실제 성과
   - 리프테라: 3대 (매우 저조)
   - 쿨소닉: 8대 (신제품 고려 시 양호)
   - 쿨페이즈: 10대 (중간)

3. 올리지오 8월 188대의 의미
   - 전체 판매 1,220대 중 15.4%
   - 8월 집중 프로모션 추정
   - 아스테라시스 전체(21대) 대비 9배

올바른 효율성 지표:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❌ 잘못된 계산: 총 판매량 / 8월 발행량
   (리프테라: 495대 / 63건 = 효율 780%)
   → 누적 판매를 월간 활동으로 나눔 (오류)
```
